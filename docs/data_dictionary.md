# 📚 Data Dictionary

## 📌 Description
This document provides details about the dataset features used in the project.

| Column Name     | Description                          |
|-----------------|--------------------------------------|
| price_egp       | Property price in Egyptian Pounds    |
| area_value      | Property area in square units        |
| location        | Property location                    |
| lat             | Latitude                             |
| lon             | Longitude                            |
| is_premium      | Premium listing flag (0/1)           |
| is_verified     | Verified listing flag (0/1)          |
| images_count    | Number of images in listing          |
| price_per_sqft  | Price per square foot (engineered)   |


## 🧠 Note
The `price_per_sqft` feature is engineered and serves as a key metric for identifying undervalued properties.
